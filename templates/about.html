ParkD

ParkD is a 36-hour long hackathon project created by
Angus Lam, Tristan Ray, Vaibhav Parashar, and Karen Xiong
at MHacks 8.

Our map finds the parking availability of parking lots around Detroit! We combined 3 APIs, Detriot Open Data, Google's Geocoding API, and Foursquare's Venues API to calculate our data, and jQuery and LeafletJS to present it. We predict what the availability is each parking lot, for each time of the day. The data for time of day can be changed by selecting 1 of the 4 time icons.


Tech Used:
HTML/CSS/JS jQuery LeafletJS
Flask Python Animate.css Bootstrap
Detroit Open Data
Google Geocoding API
Foursquare API
